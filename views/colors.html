Colors

Usage

There are several HTML and default Rexel elements that will automatically have colors applied (e.g; buttons, links, text, inputs, etc.). If an element does not have a default style, this section can be used to find the acceptable class or color hex-value.

For TEXT, add a class of rx- followed by the color name. For example; .rx-orange will output Rexel Orange.
For BACKGROUND COLOR, append the class with -bg. For example; .rx-orange-bg will output Rexel Orange.
For BORDER COLOR, append the class with -br. For example; .rx-orange-br will output Rexel Orange.
See More About Borders

Global Indicator Colors

These Colors are used only when we need to indicate something of importance to the customer about our services or products. etc. You will see these colors most often on product pages with our Indicator Icons. Each Indicator color has a both a lighter and darker hue that can be used only in conjuction with the indicator color. There are some elements that use the other shades for miscellaneous purposes. These colors should not be called on their own but are available with the customary class names.
