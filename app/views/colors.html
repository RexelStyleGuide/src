Colors

Usage

Each color class is prepended with rx- followed by the color name. For example; If you want to make some text Rexel Orange, add a class of rx-orangeto the element. If you append rx-orange with -bg, making the class rx-orange-bg, the background color of the element will use Rexel Orange. If you replace -bg with -br, making the class rx-orange-br you are now targeting the element's border-color. Rexel Orange will now show with a Rexel Orange border of 1px solid #ee7803.
